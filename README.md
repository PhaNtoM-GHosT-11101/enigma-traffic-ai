## 🔬 THE 7-STAGE CV PIPELINE

```
⚡ STAGE 1:  INGEST         ──▶  🎯 Capture & normalize camera streams (upload / RTSP)
⚡ STAGE 2:  CLAHE          ──▶  🌓 Contrast-Limited Adaptive Histogram Equalization + deblur
⚡ STAGE 3:  YOLOv8         ──▶  🧠 Detect vehicles, riders, pedestrians (YOLOv8-L)
⚡ STAGE 4:  CROP & EXTRACT ──▶  ✂️ Isolate license plates, head regions, driver torsos
⚡ STAGE 5:  POSE & FACES   ──▶  🦴 Keypoint pose estimation for helmet & seatbelt analysis
⚡ STAGE 6:  VIOLATIONS     ──▶  🚦 7-class violation engine
⚡ STAGE 7:  OCR & REPORT   ──▶  📜 Super-res plate OCR → VAHAN cross-ref → Signed PDF
```

<details>
<summary><strong>🔍 DEEP DIVE — Stage 6: Violation Engine</strong></summary>
<br>

| # | Violation | Method | Detection Logic |
|:-:|:----------|:-------|:----------------|
| 1 | **🪖 Helmet Non-Compliance** | MobileNetV3 classifier | Head-crop classification → confidence threshold |
| 2 | **🔗 Seatbelt Violation** | Pose diagonal detection | Torso keypoint line intersection with belt zone |
| 3 | **🛵 Triple Riding** | Centroid overlap | Rider + 2+ passenger bounding box centroid proximity |
| 4 | **↩️ Wrong-Side Driving** | Homography vectors | Coordinate vector direction vs lane orientation |
| 5 | **🛑 Stop-Line Violation** | Axle-Y boundary | Axle overlap of configurable Y-axis threshold line |
| 6 | **🔴 Red-Light Running** | Signal ROI entry | Vehicle entry during signal ROI red phase |
| 7 | **🚫 Illegal Parking** | Optical flow | Stationary duration exceeding parking threshold |

</details>

## 🎨 DESIGN SYSTEM — NEO-SWISS BRUTALIST

> *"Where editorial print meets surveillance command centers."*

The dashboard rejects standard dark-mode SaaS conventions. It's a **high-fidelity Neo-Swiss Brutalist / Editorial Print** aesthetic modeled after classic architectural typography and surveillance command center UIs.

### 🎭 Color Palette

```css
--paper:      #F0EDE5;   /* Off-white cream paper */
--ink:        #100C0A;   /* Deep ink typography */
--alarm:      #CC1100;   /* Warning red accents */
--caution:    #F5C518;   /* Warning yellow highlights */
```

### 🔤 Typography Stack

| Element | Font | Weight |
|:--------|:-----|:-------|
| **Headings** | `Barlow Condensed` | Heavy, condensed |
| **Body** | `Inter` | Clean, modern |
| **Code** | `DM Mono` | Monospaced precision |

### ⚡ Interaction Design

- **Cursor**: Custom crosshair state scaling on interactive elements
- **Scan Animation**: GSAP laser sweeping synchronized with bounding box flashes
- **ROI Controls**: Draggable stop-line Y & lane-divider X overlays (bi-directional with inputs)
- **Comparison Slider**: Live canvas-rendered before/after view
- **Metrics**: Real-time count-up animations (accuracy, latency, classes, scalability)
- **Evidence Modal**: Monospace court-admissible docket with SHA-256 signature
- **Audio**: Web Audio Synthesizer chirps & alarms on violation triggers
- **Splash Screen**: Animated boot sequence with progress bar

## 🚀 GETTING STARTED

### 📋 Prerequisites

- Python 3.10+
- pip package manager

### 💾 Installation

```bash
# Clone the fortress
git clone https://github.com/PhaNtoM-GHosT-11101/enigma-gridlock.git
cd enigma-gridlock

# Install artillery
pip install -r backend/requirements.txt
```

### 🎯 Launch

```bash
# Deploy the command center
./start_server.sh
```

*Or manually:*
```bash
uvicorn backend.main:app --host 127.0.0.1 --port 8000 --reload
```

Then open **http://localhost:8000** in your browser.
