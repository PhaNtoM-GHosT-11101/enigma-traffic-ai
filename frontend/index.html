// 002 — PROCESSING ARCHITECTURE

THE PIPELINE

06 STAGES / END-TO-END
01
INGEST
Camera Feed Ingestion

RTSP and HLS streams enter a distributed message queue at up to 30fps per channel. Supports live feeds and bulk batch uploads with automatic format normalization and timestamping.

02
PREPROCESS
Frame Enhancement

CLAHE contrast enhancement, motion deblur, and deep-learning rain removal ensure consistent detection quality across all weather conditions and lighting environments, 24/7.

03
DETECT
YOLOv8-L Object Detection

Fine-tuned YOLOv8-L trained on 40,000 annotated samples identifies vehicles, riders, passengers, and pedestrians at 120ms per frame on GPU hardware with sub-5% false positive rate.

04
ANALYZE
7 Parallel Sub-Detectors

Seven specialized violation modules run concurrently — helmet, seatbelt, triple-riding, wrong-side, stop-line, red-light, and illegal parking — each with independent confidence scoring.

05
RECOGNIZE
License Plate OCR + VAHAN

Super-resolution upsampling followed by multi-angle OCR achieves 97.2% plate read accuracy. Cross-referenced with the VAHAN vehicle registry for immediate owner identification.

06
EVIDENCE
Court-Admissible Records

Each violation generates a cryptographically signed, SHA-256 hashed evidence packet containing annotated frames, GPS coordinates, timestamps, and full chain-of-custody metadata.
